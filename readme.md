# Sender

Sender is a binary that helps you to send files to a selected ip.

## Usage

`sender send "file" "ip"` this is for send a file, **The selected ip has to be open with sender**, ¿How?, with
`sender get` and now with this you open the port 45434/tcp and be waiting for a file
